# Pt1.5: Docker + MongoDB - Backend d'una botiga en línia

Aquest projecte consisteix en la construcció d'un entorn de bases de dades NoSQL utilitzant MongoDB executat dins de contenidors Docker.
L'objectiu és simular el backend d'una botiga en línia, gestionant des de la configuració de l'entorn fins a la realització de consultes avançades.

## Prerequisits

Per poder replicar aquest entorn, és necessari disposar del següent programari:

* **Docker**: Versió 20.10 o superior.
* **Docker Compose**: Versió 2.0 o superior.
* **Git**: Per a la gestió del repositori.
* **WSL2 (Windows Subsystem for Linux)**: Recomanat si s'executa en entorns Windows.
